Lang / and

Logical AND operator.

Syntax:
(Boolean) [and:<(Boolean) value_1>,<(Boolean) value_2>,...]
Parameters:
values ... Operand values.
Returns:

Logical AND operation between 2 or more values.
TRUE if all values is TRUE, FALSE otherwise.

Example:
in:
[and:[true],[true]] [and:[true],[false]]
out:
true false
