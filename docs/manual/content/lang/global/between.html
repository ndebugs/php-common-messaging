Lang / between

Between operator.

Syntax:
(Boolean) [between:<(Number) value>,<(Number) min>,<(Number) max>]
Parameters:
value Source value.
min Minimum value.
max Maximum value.
Returns:

TRUE if both value is greater than or equal to min, and less than or equal to max. FALSE otherwise.

Example:
in:
[between:0,1,3] [between:1,1,3] [between:2,1,3]
out:
false true true
