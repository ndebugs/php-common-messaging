Lang / dateTime

Retrieves current date-time String.

Syntax:
(String) [dateTime:<(String) pattern>[,<(String) difference>]]
Parameters:
pattern PHP default DateTime format.
difference Unsigned Integer + unit.
Available units is: Y (year), M (month), D (day), h (hour), m (minute), s (second).
Returns:

Current date-time String from specified values.

Example:
in:
[dateTime: d-m-Y] [dateTime: d-m-Y,-1D]
out:
24-10-2014 23-10-2014
