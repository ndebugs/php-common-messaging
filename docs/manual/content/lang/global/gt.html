Lang / gt

Greater-than operator.

Syntax:
(Boolean) [gt:<(Number) value_1>,<(Number) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

TRUE if value_1 is greater than value_2, FALSE otherwise.

Example:
in:
[gt:0,1] [gt:1,0] [gt:1,1]
out:
false true false
