Lang / gte

Greater-than or equal operator.

Syntax:
(Boolean) [gte:<(Number) value_1>,<(Number) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

TRUE if value_1 is greater than or equal to value_2, FALSE otherwise.

Example:
in:
[gte:0,1] [gte:1,0] [gte:1,1]
out:
false true true
