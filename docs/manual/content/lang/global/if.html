Lang / if

Conditional selector.

Syntax:
(Object) [if:<(Boolean) condition>,<(Object) true_value>,<(Object) false_value>]
Parameters:
condition Conditional statement.
true_value True statement.
false_value False statement.
Returns:

true_value if condition is TRUE, false_value otherwise.

Example:
in:
[if:[true],TRUE,FALSE]
out:
TRUE
