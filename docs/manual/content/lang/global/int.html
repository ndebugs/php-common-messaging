Lang / int

Integer parser.

Syntax:
(Integer) [int:<(Object) value>[,<(Integer) fromBase>[,<(Integer) toBase>]]]
Parameters:
value Source value.
fromBase Base input number of value, default value is 10.
toBase Base output number of value, default value is 10.
Returns:

Integer from specified values.

Example:
in:
[int:123] [int:ff,16] [int:255,10,16]
out:
123 255 ff
