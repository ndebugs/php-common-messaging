Lang / isEmpty

Empty-equal operator.

Syntax:
(Boolean) [isEmpty:<(Object) value>]
Parameters:
value Source value.
Returns:

TRUE if value is empty, FALSE otherwise.

Example:
in:
[isEmpty:[null]] [isEmpty:0] [isEmpty:] [isEmpty:abc]
out:
true true true false
