Lang / isNull

NULL-equal operator.

Syntax:
(Boolean) [isNull:<(Object) value>]
Parameters:
value Source value.
Returns:

TRUE if value is NULL, FALSE otherwise.

Example:
in:
[isNull:[null]] [isNull:0] [isNull:] [isNull:abc]
out:
true false false false
