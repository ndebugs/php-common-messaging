Lang / lt

Less-than operator.

Syntax:
(Boolean) [lt:<(Number) value_1>,<(Number) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

TRUE if value_1 is less than value_2, FALSE otherwise.

Example:
in:
[lt:0,1] [lt:1,0] [lt:1,1]
out:
true false false
