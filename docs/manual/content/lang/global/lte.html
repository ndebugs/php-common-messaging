Lang / lte

Less-than or equal operator.

Syntax:
(Boolean) [lte:<(Number) value_1>,<(Number) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

TRUE if value_1 is less than or equal to value_2, FALSE otherwise.

Example:
in:
[lte:0,1] [lte:1,0] [lte:1,1]
out:
true false true
