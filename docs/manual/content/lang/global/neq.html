Lang / neq

Not-equal operator.

Syntax:
(Boolean) [neq:<(Object) value_1>,<(Object) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

TRUE if value_1 is not equal to value_2, FALSE otherwise.

Example:
in:
[neq:abc,abc] [neq:abc,def]
out:
false true
