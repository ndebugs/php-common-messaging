Lang / notEmpty

Non-empty selector.

Syntax:
(Object) [notEmpty:<(Object) value_1>,<(Object) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

Non empty value between value_1 and value_2, or NULL if both values is empty.

Example:
in:
[notEmpty:abc,def] [notEmpty:,def]
out:
abc def
