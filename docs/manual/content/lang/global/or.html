Lang / or

Logical OR operator.

Syntax:
(Boolean) [or:<(Boolean) value_1>,<(Boolean) value_2>,...]
Parameters:
values ... Operand values.
Returns:

Logical OR operation between 2 or more values.
FALSE if all values is FALSE, TRUE otherwise.

Example:
in:
[or:[true],[true]] [or:[true],[false]]
out:
true true
