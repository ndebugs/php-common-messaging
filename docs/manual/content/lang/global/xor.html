Lang / xor

Logical XOR operator.

Syntax:
(Boolean) [xor:<(Boolean) value_1>,<(Boolean) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

Logical XOR operation between value_1 and value_2.
TRUE if value_1 is not equal to value_2, FALSE otherwise.

Example:
in:
[xor:[true],[true]] [xor:[true],[false]]
out:
false true
