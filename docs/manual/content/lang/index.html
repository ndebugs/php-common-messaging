Lang

One-liner language to support messaging.

Syntax:
[[<package> ]<name>[:<parameter>,...]]
Reserved Characters:
[ Prefix.
] Suffix.
: Parameter prefix.
, Parameter delimiter.
\ Escape character.

Package

/ Global package.
/string/ Contains String operations.
/math/ Contains arithmetic operations.
/list/ Contains List operations.
/map/ Contains Map operations.
/message/ Contains operations to current Message.
