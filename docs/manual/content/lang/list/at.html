Lang / list / at

Retrieves an Object from a List.

Syntax:
(Object) [list at:<(List) source>,<(Integer) index>]
Parameters:
source Source value.
index Unsigned Integer, indicates index. Start from 0 to source length - 1.
Returns:

Retrieved value from source at specified index.

Example:
in:
[list at:[list:abc,def,ghi],0]
out:
abc
