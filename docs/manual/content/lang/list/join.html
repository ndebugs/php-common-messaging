Lang / list / join

Joins all List values into a String.

Syntax:
(String) [list join:<(List) value>,<(String) delimiter>]
Parameters:
value Source value.
delimiter Target delimiter.
Returns:

String from joined value with specified delimiter between its elements.

Example:
in:
[list join:[list:abc,def,ghi],-]
out:
abc-def-ghi
