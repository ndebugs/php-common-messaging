Lang / map / contains

Evaluates if a Map values contains an Object.

Syntax:
(Boolean) [map contains:<(Map) source>,<(Object) value>]
Parameters:
source Source value.
value Target value.
Returns:

TRUE if source contains value, FALSE otherwise.

Example:
in:
[map contains:[map:[entry:a,123],[entry:b,456],[entry:c,789]],123]
out:
true
