Lang / math / min

Minimum Number selector.

Syntax:
(Number) [math min:<(Number) value_1>,<(Number) value_2>]
Parameters:
value_1 Left operand value.
value_2 Right operand value.
Returns:

Minimum value between value_1 and value_2.

Example:
in:
[math min:0,1] [math min:1,1]
out:
0 1
