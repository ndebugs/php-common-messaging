Lang / string / at

Retrieves a String from a String.

Syntax:
(String) [string at:<(String) source>,<(Integer) index>,[<(Integer) length>]]
Parameters:
source Source value.
index Signed Integer, indicates start index.
Start from 0 to source length - 1, or -1 to source length * -1 for reversed substring.
length Unsigned Integer, indicates length.
Returns:

Substring of source from specified index and length, or until the end of source if length is NULL.

Example:
in:
[string at:abcdef,0,3] [string at:abcdef,3] [string at:uvwxyz,-4] [string at:uvwxyz,-1,3]
out:
abc def uvw xyz
