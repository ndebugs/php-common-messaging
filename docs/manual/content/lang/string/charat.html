Lang / string / charAt

Retrieves a Character from a String.

Syntax:
(Character) [string charAt:<(String) source>,<(Integer) index>]
Parameters:
source Source value.
index Unsigned Integer, indicates index. Start from 0 to source length - 1.
Returns:

Retrieved Character of source from specified index.

Example:
in:
[string charAt:abc,1]
out:
b
