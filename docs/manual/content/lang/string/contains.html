Lang / string / contains

Evaluates if a String contains another String.

Syntax:
(Boolean) [string contains:<(String) source>,<(String) value>]
Parameters:
source Source value.
value Target value.
Returns:

TRUE if source contains value, FALSE otherwise.

Example:
in:
[string contains:abc,a]
out:
true
