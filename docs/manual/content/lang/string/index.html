Lang / string /

Contains String operations.

at Retrieves a String from a String.
charAt Retrieves a Character from a String.
length Retrieves size of a String.
contains Evaluates if a String contains another String.
lowCase Converts all Characters of a String to lower case.
upCase Converts all Characters of a String to upper case.
pad Pads a String.
trim Trims a String.
lTrim Trims only left side of a String.
rTrim Trims only right side of a String.
replace Replaces some Characters of a String.
match Matches a String with a Regular Expression.
split Splits a String to a List.
