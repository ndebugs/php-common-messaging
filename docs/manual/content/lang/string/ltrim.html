Lang / string / lTrim

Trims only left side of a String.

Syntax:
(String) [string lTrim:<(String) value>[,<(String) characters>]]
Parameters:
value Source value.
characters Trimmed character(s).
Returns:

Same as trim, but only processed left side of value.

Example:
in:
[string lTrim:  ABC def  ] [string lTrim:000123,0] [string lTrim:456000,0]
out:
ABC def   123 456000
