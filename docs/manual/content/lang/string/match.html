Lang / string / match

Matches a String with a Regular Expression.

Syntax:
(String) [string match:<(String) value>,<(String) pattern>]
Parameters:
value Source value.
pattern Regular Expression pattern.
Returns:

TRUE if value matches pattern, FALSE otherwise.

Example:
in:
[string match:abc123,^\[a-zA-Z0-9\]*$]
out:
true
