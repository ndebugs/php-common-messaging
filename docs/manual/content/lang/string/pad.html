Lang / string / pad

Pads a String.

Syntax:
(String) [string pad:<(String) value>,<(Character) character>,<(Integer) length>]
Parameters:
value Source value.
character Pad character.
length Signed Integer, indicates length and padding direction. Signed length means padding from right to the left.
Returns:

Padded value with specified character and length.

Example:
in:
[string pad:123,0,-6] [string pad:456,0,6]
out:
000123 456000
