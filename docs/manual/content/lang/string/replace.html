Lang / string / replace

Replaces some Characters of a String.

Syntax:
(String) [string replace:<(String) source>,<(String) pattern>,<(String) value>]
Parameters:
source Source value.
pattern Regular Expression pattern.
value Replacement.
Returns:

String replacement of source with value by specified pattern.

Example:
in:
[string replace:A1B3C1 1d3e3f,\[13\],]
out:
ABC def
