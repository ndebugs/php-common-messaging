Lang / string / rTrim

Trims only right side of a String.

Syntax:
(String) [string rTrim:<(String) value>[,<(String) characters>]]
Parameters:
value Source value.
characters Trimmed character(s).
Returns:

Same as trim, but only processed right side of value.

Example:
in:
[string rTrim:  ABC def  ] [string rTrim:000123,0] [string rTrim:456000,0]
out:
  ABC def 000123 456
