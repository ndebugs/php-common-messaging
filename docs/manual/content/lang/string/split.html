Lang / string / split

Splits a String to a List.

Syntax:
(String) [string split:<(String) value>,<(String) delimiter>]
Parameters:
value Source value.
delimiter Source delimeter.
Returns:

List from extracted value separated by delimiter.

Example:
in:
[string split:1-2-3,-]
out:
[1,2,3]
