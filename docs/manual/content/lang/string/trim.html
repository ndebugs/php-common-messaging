Lang / string / trim

Trims a String.

Syntax:
(String) [string trim:<(String) value>[,<(String) characters>]]
Parameters:
value Source value.
characters Trimmed character(s).
Returns:

Trimmed String of value by specified characters. If characters is NULL, then whitespace characters is used.

Example:
in:
[string trim:  ABC def  ] [string trim:000123,0] [string trim:456000,0]
out:
ABC def 123 456
