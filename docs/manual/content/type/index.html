Type

Messaging data-types.

Syntax:
<name>[(<parameter>,...)]
Reserved Characters:
( Parameter prefix.
) Parameter suffix.
, Parameter delimiter.
\ Escape character.

Object

Null Contains NULL value.
Boolean Contains Java Boolean value.
Character Contains Java Character value.
String Contains Java String value.
Entry Contains key and value, used as Map value.

Number

Integer Contains Java default Integer format.
Decimal Contains Java default Double format.

Collection

List Contains Java Array value.
Map Contains Java Map value.
